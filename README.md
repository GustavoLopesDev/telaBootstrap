# Projeto Bootstrap de Estudos

Este projeto tem como objetivo **praticar HTML, CSS e o framework Bootstrap**. Ele **não possui lógica JavaScript ou backend**, sendo totalmente focado na construção de interfaces responsivas e componentes visuais com Bootstrap 5.

## 🔍 Navegação

- A **Página 1** pode ser acessada ao clicar no link **"Faça Login"** na tela inicial.
- As demais páginas estão interligadas por **botões de navegação no footer**, permitindo ir de uma para outra com facilidade.
- Todo o layout foi estruturado para ser **totalmente responsivo**, funcionando bem em dispositivos móveis e desktops.
